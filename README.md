# 📊 Netflix India YouTube Channel Analysis

## 🔍 Project Overview
This project analyzes Netflix India’s YouTube channel to understand video performance, engagement, and content strategy.

## 🛠 Tools Used
- Python (Pandas, Matplotlib)
- Data Cleaning & Feature Engineering
- Exploratory Data Analysis

## 📈 Key Insights
- Long-form videos generate higher views and engagement  
- Strong positive relationship between views and comments  
- Evening (5–9 PM) uploads perform better  
- Trailer and series-based content drive higher performance  
- Tags improve discoverability, but content quality is the main driver  
- Views follow a viral distribution pattern (few videos drive most views)


  ## 💡 Business Recommendations  
- Focus on high-quality long-form content  
- Optimize publishing time for peak engagement  
- Use relevant and branded tags  
- Leverage trending topics and celebrity-based content  
- Identify and replicate viral content patterns

This project demonstrates how data analytics can be used to derive actionable insights and improve content strategy in a real-world business context.
